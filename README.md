vagrant-wls12.1.2-coherence-mysql
=================================

creates a patched 12.1.2 WebLogic cluster ( adminwls,nodewls1 , nodewls2 ) with a MySQL database

weblogic 12.1.2
- jdk-7u45-linux-x64.tar.gz
- wls_121200.jar
- p16175470_121200_Generic.zip

# admin server  
vagrant up adminwls

# node1  
vagrant up nodewls1

# node2  
vagrant up nodewls2
